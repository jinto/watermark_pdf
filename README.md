pdf watermarking with text
==========================

little code for watermarking on the pdf file, with text.


Compile
=======
javac -cp itextpdf-5.4.3.jar watermark.java

Usage
====
java -cp itextpdf-5.4.3.jar:./ watermark source.pdf target.pdf "licensed user"
